# diarios

A python package to extract information from Brazilian official diaries

## Requirements

- Python 3.6 or 3.8.6
- The Python packages in [requirements.txt](requirements.txt)
- [pcre2grep](https://www.pcre.org/) (for extract.py)

## Installation

To install run e.g.

```bash
python3 -m pip install git+ssh://git@github.com/hsigstad/diarios.git --user
```

or clone de repo and run

```bash
pip install -e ~/diarios`
```
